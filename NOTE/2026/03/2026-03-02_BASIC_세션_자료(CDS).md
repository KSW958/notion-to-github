Level 1: 항공사별 운항 횟수 집계 (입문)



1. Basic (ZI_CarrierBase): 필드명 변경

```javascript
define view entity ZI_CarrierBase as select from scarr {
  key carrid as CarrierID,
      carrname as CarrierName
}
```



2. Composite (ZI_FlightStats): 운항 횟수 집계 (Aggregation)

```javascript
define view entity ZI_FlightStats as select from sflight {
  key carrid as CarrierID,
      count(*) as TotalFlightCount
} group by carrid
```



3. Consumption (ZC_CarrierReport): 최종 결합

### 1. [기존 로직 유지] 집계 뷰 (ZI_FlightStats)

이건 우리가 처음에 만든 운항 횟수 집계 로직 그대로입니다. (재사용)



```javascript
@EndUserText.label: '항공사별 운항 횟수 집계'
define view entity ZI_FlightStats 
  as select from sflight 
{
  key carrid as CarrierID,
      count(*) as TotalFlightCount,
      /* 💡 추가: 항공사별 평균 점유율 로직만 살짝 덧붙임 */
      cast( avg( seatsocc ) as abap.dec(10,2) ) as AvgOccupancy
} group by carrid
```
